Reflection

I have used several elements and materials that I have learned so far to create this project.
The most challenging part within this project is the layout and positioning. I am really proud of
myself for making this mobile screen responsive design. I was always exposed to resources that was mainly
builded for desktop screens or medium screens such as tablets.
I have used <img> tag to include the pictures that I have chosen for this project.
This reason why I chose this specific tag is that this was the most suited for grid structured container.
The best part of this all is when everything finally work out & you are able to implement your idea..
In order to build this website I have gone through all the learning materials
that were available on the university platform. The resources included several
links into tutorial websites that would be beneficial for designing.
It took huge amount of time, effort & patience to create this project but luckily we are two members this time.
Going back to the challenges that I had while creating this project, I would say that positioning can be really complicated.
This is especially when you do not design lots of pages yet..
I think the grid structures are the most beneficial solution for website design..
I have always thought that designing is very creative and fun but time challenging as well.
I look forward to work with such website designing projects in the future . .
